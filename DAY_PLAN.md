## 🎒 ก่อนเดินทาง (เช็กที่บ้านให้ครบ)
- [ ] โน้ตบุ๊ก Core i5+ + **ที่ชาร์จ** + **ปลั๊กต่อพ่วง**
- [ ] ติดตั้งไลบรารีครบ และ **รัน `python demo_gui.py` + `--listen` 1 ครั้งตอนมีเน็ต** (ให้ Whisper โหลดเสร็จ → หน้างานออฟไลน์ได้)
- [ ] ก๊อปทั้งโฟลเดอร์โปรเจกต์ + ไฟล์ `.pkl` ลง **USB สำรอง** (เผื่อโน้ตบุ๊กมีปัญหา)
- [ ] เตรียม **คลิปเสียงมิจฉาชีพ 2–3 คลิป** ไว้เป็นแผนสำรองตอนเดโม
- [ ] **ซ้อมพูดตาม [PITCH_SCRIPT.md](PITCH_SCRIPT.md)** แบ่งบท A/B/C จับเวลา ≥ 3 รอบ
- [ ] เช็กกับครูเรื่อง **แผน depa Infrastructure Fund (15 คะแนน)** เตรียมหรือยัง
- [ ] เตรียมการแต่งกาย: นักเรียน = ชุดนักเรียน/ชุดช็อป/ชุดพละ · ครู = ชุดสุภาพ

- [ ] มีส่วนร่วม ตั้งใจ ทำงานเป็นทีม (= คะแนน Participation)

## 🟥 DAY 3 (23 มิ.ย.) — Pitching  → นักเรียนพูดเอง 7 นาที + ตอบ 3 นาที
**เป้าหมายวันนี้:** นำเสนอให้ปัง — **ครูช่วยไม่ได้เลย**

| เวลา | สิ่งที่ต้องทำ |
|------|--------------|
| **08.00** | ลงทะเบียน |
| 08.30–12.00 | เริ่มนำเสนอ (ทีมละ 7 นาที + ถาม-ตอบ 3 นาที) |
| 13.00–16.30 | นำเสนอต่อ |
| 17.30 | จบกิจกรรม |

**ก่อนถึงคิว:**
- [ ] เปิดโน้ตบุ๊ก + ต่อบอร์ด + **เปิด Whisper ค้างไว้** (ไม่ต้องรอโหลดตอนขึ้นเวที)
- [ ] เตรียม **เดโม 2 เคสที่ชัวร์** (1 หลอก, 1 ปกติ) + แผนสำรอง (โหมดพิมพ์/คลิปเสียง)
- [ ] เช็กไมค์ + ระดับเสียงห้อง

**ระหว่างนำเสนอ:**
- [ ] เปิดด้วย **ปัญหา** → เฉลยชื่อ **SAFE** → ทางแก้ → **สาธิตสด** → ความแม่นยำ → ผลกระทบ → ปิด
- [ ] พูดช้า ชัด สบตากรรมการ แบ่งบท A/B/C ตามซ้อม
- [ ] ตอนถาม-ตอบ ใครถนัดเรื่องไหนตอบเรื่องนั้น (เตรียมแนวตอบจาก [TEAM_GUIDE.md](TEAM_GUIDE.md) §7)
