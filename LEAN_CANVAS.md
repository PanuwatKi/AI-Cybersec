# 🧩 Lean / Idea Canvas — AI คัดกรองสายมิจฉาชีพ 3 สี

ใช้ส่งกรรมการ (ส่วน Problem & Value) และเตรียมตอน Day 1 ก่อนขึ้น Prototype

| ช่อง | รายละเอียด |
|------|-----------|
| **1. ปัญหา (Problem)** | • คนไทยถูกแก๊งคอลเซ็นเตอร์หลอกสูญเงินปีละ >2 หมื่นล้านบาท<br>• ผู้สูงอายุแยกไม่ออกว่าสายไหนมิจฉาชีพ<br>• มิจฉาชีพใช้จิตวิทยากดดันให้รีบโอนก่อนได้คิด |
| **2. กลุ่มเป้าหมาย (Customer Segments)** | • หลัก: ผู้สูงอายุที่อยู่บ้านลำพัง<br>• รอง: คนทั่วไป, ลูกหลานที่อยากปกป้องพ่อแม่<br>• ขยายผล: ศูนย์ผู้สูงอายุ/ชุมชน/อปท. |
| **3. คุณค่าที่ส่งมอบ (Unique Value Proposition)** | "ผู้ช่วยเตือนภัยมิจฉาชีพแบบเรียลไทม์ ด้วยไฟ 3 สีที่เข้าใจง่าย ทำงานออฟไลน์ ปลอดภัยต่อความเป็นส่วนตัว" |
| **4. ทางแก้ (Solution)** | ฟังบทสนทนา → AI ถอดเสียงเป็นข้อความ (Whisper) → วิเคราะห์ความเสี่ยง (TF-IDF + Naive Bayes) → เตือนด้วยไฟ 🔴🟡🟢 + % + เสียง Buzzer |
| **5. ช่องทางเข้าถึง (Channels)** | กล่องอุปกรณ์ตั้งข้างโทรศัพท์บ้าน, ต่อยอดเป็นแอปมือถือ, แจกจ่ายผ่านหน่วยงานดูแลผู้สูงอายุ |
| **6. รายได้/คุณค่าทางสังคม (Revenue / Value)** | ลดความสูญเสียทางการเงินของสังคม; โมเดลพัฒนาต่อ: ขายอุปกรณ์ราคาประหยัด/บริการ subscription อัปเดตฐานข้อมูล/ความร่วมมือภาครัฐ |
| **7. โครงสร้างต้นทุน (Cost Structure)** | บอร์ดไมโครคอนโทรลเลอร์ราคาถูก + เซนเซอร์ไฟ/เสียง; ซอฟต์แวร์โอเพนซอร์ส; ต้นทุนหลักคือการดูแล/อัปเดตข้อมูล |
| **8. ตัวชี้วัด (Key Metrics)** | ความแม่นยำของโมเดล (ปัจจุบัน ~94% CV), จำนวนสายที่เตือนถูก, จำนวนผู้ใช้, มูลค่าความเสียหายที่ป้องกันได้ |
| **9. จุดแข็งที่เลียนแบบยาก (Unfair Advantage)** | วิเคราะห์ "เนื้อหาบทสนทนา" ไม่ใช่แค่บล็อกเบอร์ → จับสคริปต์/เบอร์ใหม่ได้; ทำงานออฟไลน์ 100%; ทนต่อการถอดเสียงเพี้ยน; ฐานข้อมูลภาษาไทยที่ขยายได้ |
