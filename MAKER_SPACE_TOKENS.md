# 🪙 รายการ Token (Maker Space) + แผนเบิกของ — SAFE

> ทุกทีมมี **1,000 Token** · ใช้ **0–800 (≤80%) = ได้คะแนนเต็ม 5** · ของแต่ละรายการ **ขายเป็นชุด แยกไม่ได้**
> ⚠️ คะแนน Token เต็มได้ต่อเมื่อ Functionality อยู่ระดับ **ดี–ดีมาก** (ถ้าต่ำกว่าจะถูกลดเหลือ 2)

## ✅ ของที่ได้ฟรี (ไม่เสีย Token)
**Arduino UNO Q, Modulino ทุกโมดูล (Pixels/Buzzer/ฯลฯ), ไมโครโฟน** — ผู้จัดเตรียมให้

👉 **โปรเจกต์ SAFE ใช้แค่ของฟรีทั้งหมดในส่วนอิเล็กทรอนิกส์:**
ไฟ 3 สี = **Modulino Pixels (ฟรี)** · เสียงเตือน = **Modulino Buzzer (ฟรี)** · รับเสียง = **ไมค์ (ฟรี)**
→ **ไม่ต้องซื้อ RGB LED (30) หรือ buzzer (45) แยก** = ประหยัด Token (เป็นจุดที่พูดได้ว่า "ออกแบบคุ้มค่า")

## 🛒 แผนเบิกที่แนะนำ (ใช้แค่ทำกล่อง/ฐานให้ดูเป็นผลิตภัณฑ์)
| ตัวเลือก | รายการ | รวม Token |
|---------|--------|----------:|
| **A — ดูเรียบหรู** | plastwood (80) + cable tie (25) + เทปพันสายไฟ (35) | **140** |
| **B — ประหยัดสุด** | กระดาษฟูก A4 (20) + ไม้ไอติมใหญ่ (35) + cable tie (25) | **80** |

ทั้งสองแบบ **ต่ำกว่า 800 มาก → ได้ Token เต็ม 5 สบาย ๆ** เหลือ Token เผื่อซื้อเพิ่มถ้าจำเป็น
